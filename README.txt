(영어)

Hello! Our web-based demo runs on Python. To run it, simply follow the steps below!

1. Install Python:
If Python isn't installed locally, go to https://www.python.org/downloads/ and download the latest version of Python.
When installing, be sure to add "Add Python.exe to PATH"! This is important. Then, click "Install Now."

2. Install Flask:
Once Python is installed, press Windows + R or press the Windows key, then search for "CMD" in the search bar to open the "Command Prompt."
Enter the following in the command prompt:
pip install flask
Once the installation is complete, close the CMD window.

3. Open the bat file:
Double-click the STARTAPP.bat file to run it. Then, hold down the Control key and click the URL http://127.0.0.1:5000.
This will launch the webpage we created locally!

(한국어)

안녕하세요! 우리 웹기반의 데모는 Python 기반으로 동작하며 데모를 실행하기 위해서는 아래와 같은 단계를 따르시면 됩니다!

1. Python 설치 : 
만약 로컬에 Python이 설치돼 있지 않다면, https://www.python.org/downloads/ 에 들어가서 최신 버전의 Python을 다운로드 해주세요.
설치할 때, Add Python.exe to PATH에 추가해주세요! 중요합니다. 그리고 Install Now를 눌러주세요. 

2. Flask 설치 : 
Python 설치가 완료되었다면, 윈도우+R을 누르거나 윈도우 키를 눌러 뜨는 검색창에 CMD를 검색해서 나오는 "명령 프롬프트"를 열어주세요.
그리고 명령창에 다음을 입력해주세요!
pip install flask
설치가 완료됐다면 CMD 창을 종료해주세요.

3. Bat 파일 열기 : 
STARTAPP.bat 파일을 더블 클릭해서 실행해주세요. 그리고 출력되는 http://127.0.0.1:5000라는 주소를 컨트롤 키를 누른 채 클릭해 접속해주세요.
그럼 로컬에서 저희가 만든 웹페이지가 실행됩니다!
